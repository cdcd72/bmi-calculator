# BMI 計算機規格

1. 一個輸入框，用來輸入身高
2. 一個輸入框，用來輸入體重
3. 一個按鈕，用來計算 BMI
4. 一個標籤，用來顯示實際 BMI 數值
5. BMI 數值旁，再顯示一個基準標籤，請參考下表

   | BMI 數值        | 基準標籤 |
   | --------------- | -------- |
   | BMI< 18.5       | 過輕     |
   | 18.5 ≦ BMI < 24 | 正常體重 |
   | 24 ≦ BMI < 27   | 過重     |
   | BMI ≧ 27        | 肥胖     |
